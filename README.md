<h1>Portfolio Lukas Hecht</h1>
Hello, my name is Lukas Hecht. I’m a graphic designer and digital creator working across visual identity, digital design, and motion.

This repository contains the source code for my personal portfolio website. The site presents selected projects and serves as a platform to communicate my design approach, aesthetic direction, and technical capabilities.

<h2>About</h2>
My work focuses on creating clear, distinctive, and adaptable visual systems. I’m particularly interested in the relationship between identity, transformation, and digital space and how design can create trust, structure and recognition.

The website is designed to reflect this approach:
- Structured but flexible
- Minimal yet expressive
- System-based and scalable

<h2>Services</h2>

- Visual Identity
- Digital Design
- Motion Design
- Editorial & Typography
- Film & Moving Image
- Concept Development

<h2>Technical Overview</h2>
The website is built with a lightweight and modular structure:

- HTML5
- CSS3 (flexbox-based layout system)
- JavaScript for interactive elements
- Responsive design principles
- Optimized media formats for performance

Development and refinement were supported by AI-assisted tools for research, optimization and debugging.
